# Translations Policy

HoverCraft should remain compatible with WordPress translation standards, including the `hovercraft` text domain and normal WordPress translation loading behavior.

The theme may include a `languages` directory for compatibility, but HoverCraft does not maintain bundled translation files as part of normal development.

User-facing strings should remain translation-ready where practical by using standard WordPress escaping and translation functions.

Do not add large bundled `.po` or `.mo` translation files to the theme unless there is a specific release reason.
